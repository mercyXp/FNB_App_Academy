## Paragraphs in HTML

Paragraphs in an HTML document are defined by the `<p>` tag.

## Example Code

```html
<!DOCTYPE html>
<html>
<head>
</head>
<body>
  <p>Hello everyone</p>
  <p>This is fun</p>
</body>
</html>
